ความสามารถ

C++ :(ระดับพื้นฐาน) มีความเข้าใจและสามารถเขียนโค้ดคำสั่งพื้นฐานได้ เช่น การจัดการตัวแปร การควบคุมเงื่อนไข และการใช้งานฟังก์ชันเบื้องต้น

PHP : (ระดับกลาง)มีความสามารถในการพัฒนาเว็บไซต์ด้วย PHP โดยสามารถเขียนโปรแกรมที่ซับซ้อนและจัดการการสื่อสารระหว่างเซิร์ฟเวอร์และฐานข้อมูลได้เป็นอย่างดี

Python : (ระดับพื้นฐาน)เข้าใจหลักการและโครงสร้างของ Python สามารถเขียนโปรแกรมพื้นฐานและใช้ไลบรารีที่ง่ายต่อการจัดการข้อมูลได้

HTML : (ระดับพื้นฐาน)มีความเข้าใจในการจัดโครงสร้างหน้าเว็บ สามารถสร้างหน้าเว็บพื้นฐานและใช้แท็ก HTML สำหรับการจัดเรียงข้อมูลได้

MySQL : (ระดับกลาง)มีความสามารถในการสร้างและจัดการฐานข้อมูล SQL สามารถออกแบบฐานข้อมูล เขียนคำสั่ง CRUD (Create, Read, Update, Delete) และสร้างคำสั่ง SQL ที่ซับซ้อนได้

ทักษะการใช้งานโปรแกรม

Microsoft Office :(ระดับกลาง) มีความเชี่ยวชาญในการใช้งาน Microsoft Office (เช่น Word, Excel, PowerPoint) สามารถจัดการเอกสาร สร้างตารางวิเคราะห์ข้อมูล และนำเสนอข้อมูลได้อย่างมืออาชีพ

Photoshop : (ระดับกลาง) มีทักษะในการใช้งาน Adobe Photoshop สำหรับการออกแบบและปรับแต่งรูปภาพ สามารถสร้างงานกราฟิกที่สวยงามและเป็นมืออาชีพ ใช้เครื่องมือที่ซับซ้อนเพื่อสร้างภาพที่มีคุณภาพ

Illustrator : (ระดับพื้นฐาน)มีความเข้าใจในหลักการพื้นฐานของ Adobe Illustrator สามารถสร้างกราฟิกเวกเตอร์ที่เรียบง่าย และใช้งานเครื่องมือเบื้องต้นในการออกแบบได้

ChatGPT : (ระดับพื้นฐาน)มีความสามารถในการใช้งาน ChatGPT ในการค้นหาข้อมูล การตอบคำถาม และการสนับสนุนงานเขียนหรือการทำงานในเชิงข้อมูล สามารถใช้งานได้ในระดับเริ่มต้นและเข้าใจหลักการทำงานเบื้องต้นของระบบ AI ช่วยในการประมวลผลและให้ข้อมูลตามความต้องการ
